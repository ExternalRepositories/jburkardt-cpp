BMP_TO_PPMA
BMP => ASCII Portable Pixel Map (PPM) Graphics File Conversion

BMP_TO_PPMA is a C++ program which converts a Microsoft BMP file to an ASCII Portable Pixel Map (PPM) file.

Usage:

bmp_to_ppma file.bmp file.ppm
reads data from file.bmp and creates an equivalent graphics file in file.ppm.
