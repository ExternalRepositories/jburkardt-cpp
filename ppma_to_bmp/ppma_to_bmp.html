PPMA_TO_BMP
PPMA => BMP Graphics File Conversion

PPMA_TO_BMP is a C++ program which converts an ASCII Portable Pixel Map (PPM) file to a Microsoft BMP file.

Usage:

ppma_to_bmp file.ppma file.bmp
reads data from file.ppma and creates an equivalent graphics file in file.bmp.
