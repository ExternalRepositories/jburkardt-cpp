PPMB_TO_BMP
PPMB => BMP Conversion

PPMB_to_BMP is a C++ program which converts a binary PPM file to a Microsoft BMP file.

Usage:

ppmb_to_bmp file.ppmb file.bmp
reads data from file.ppmb and creates an equivalent graphics file in file.bmp.
